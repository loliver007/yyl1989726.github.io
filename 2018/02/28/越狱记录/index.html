重新越狱需要安装的东西

  1. Saigon 重新越狱 重启
  2. Cydia 下载

USB连接

  1. cd USBSSH/usbmuxd-1.0.8/python-client
  2. python tcprelay.py -t 22:2222 //把本地2222端口转发到iOS的22端口
  3. ssh root@localhost -p 2222 // ssh 到 iOS中

ps -e | grep Youkui4Phone.app

启动LLDB

  1. USB连接
  2. debugsever 附加到对应的应用 debugserver *:1234 -a “应用名” 例如：debugserver *:1234 -a “MobileNotes”
  3. 把本地1234端口转发到iOS 1234端口 python tcprelay.py -t 1234:1234
  4. 用lldb开始调试
    /Applications/Xcode.app/Contents/Developer/usr/bin/lldb
    process connect connect://localhost:1234

方法断点内存地址

根据IDA获取 方法基址 以及模块基址
根据LLDB获取模块偏移值 指令：image list -o -f 能获取模块的ASLR值 以及偏移后的模块的虚拟内存地址
公式1：
偏移后模块基地址 = 偏移前模块基地址 + ASLR偏移
偏移后符号基地址 = 偏移前符号基地址 + 符号所在模块ASLR偏移
偏移后指令基地址 = 偏移前指令基地址 + 指令所在模块ASLR偏移

方法2 lldb
1、LLDB连接到程序
2、找到需要下断点的类，在LLDB命令行输入po [类名 _shortMethodDescription]
